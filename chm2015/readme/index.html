Instructions

Competitor instructions

You will be tested on your ability to develop a website running on a web server and displayed on a web browser to a minimum HTML5 standard. You are expected to have a professional attitude and general good disposition.

It is suggested that you read carefully through this assignment and look through the provided content for the website such as the images and text.

Rules & rating

Resources: Do not bring in any resources or use the Internet. The competition has to be completed with the resources provided.

Documentations: You have access to the language documentations by using the offline documentation browser Zeal. To run the application execute ZealPortable.exe under C:\xampp\htdocs\sm2015\material\ZealPortable.

Skills and material: You are free to produce your material with any skills that you possess and use any of the available tools. In addition to the material provided it is of course up to you, if you want to produce new material or change the existing one where you feel it is necessary.

Working time: Probably there will not be enough time to complete all formal and technical criteria.

Work order: You are free to choose an own working order. We suggest that you start with the technology you know best.

Work folder: The work folder is located under C:\xampp\htdocs\sm2015\work

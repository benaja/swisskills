Part three - HTML / CSS skills

Total points: 64

General information

In part three you will prove your HTML and CSS programming skills. Therefore you will modify several prepared HTML and CSS files. Your working space is the folder work/part_three/.

You can only change or create new files under work/part_three/task1/ and work/part_three/task2/. All other files will be reverted for the review process.

Task 1 - Understanding of existing code

Your first task is to fix the syntax and logical errors in the code of task 1.

Follow the instructions in the code comment blocks. Only fixes following the instructions contained in the @todo description are accepted and will be rated.

Errors of the same kind are counting as one error.
  • HTML errors: Fix the four errors in the given HTML file:
    • task 1 - fix all html errors - work/part_three/task1/index.html
  • CSS errors: Fix the five errors in the given CSS file:
    • task 1 - fix all css errors - work/part_three/task1/style.css

  • Expected result

    All syntax and logical errors are solved.

Marking scheme

Objective 10

  • 5 All five HTML errors are fixed.
    (1 point for each error solved)
  • 5 All five CSS errors are fixed.
    (1 point for each error solved)

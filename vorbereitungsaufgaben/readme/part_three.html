Task 1 - Website template

You got a PSD from your designer, which you have to create a template from. He also provided you the used font.

PSD: The zip file (containing the PSD file) you can find in the material folder: material/part_three/secret_image/design.zip. The password will be distributed some time of the day (only at the real competition, in the example the file is not protected).

Create a website template (HTML, CSS) from scratch, according to the given PSD file. Place all files in the work/part_three/ folder.

The design is divided into 7 sections. You can skip one section (except header or footer) without any points penalty.

Font: You have to use the webfont provided. You can find the file on the material page.

bonus Images: You can get bonus points by using pictures in SVG format.

bonus Responsive: The customer asked you to also propose how the website should look like on smaller screens. You can get extra points for making the website responsive too.

  • Expected result

    The website templating (HTML, CSS and image files), placed under work/part_three/

Marking scheme

Templating 42

  • 6 Template: A static template for the given page is provided (HTML, CSS, Images, ...). (3) Only content is in the HTML-file, all design and layout aspects are handled in the CSS-files for display (3).
  • 10 Implementation: The website uses the same colors (2), font sizes (2) and spacings (2) as given in the PSD. The page looks similar to the design using Google Chrome (2) and does not fall apart on different viewport sizes (2).
  • 6 Sections: 6 out of the 7 sections are implemented (6 - one for each section).
  • 3 Webfont: The webfont provided is used (1) and embedded using CSS (2).
  • 4 Compliance: HTML (2) and CSS (2) are W3C compliant (no errors, half point reduction for each error).
  • 6 Accessibility: The website can also be used with other tools than regular browsers (page language, title, meta-tags for contenttype, author, keywords and description, title/alt attributes, no tables for layout, no frames) (4). Structure is also suitable without CSS (2).
  • 7 bonus Extra: Images used are saved in SVG format (SVG: 2, PNG: 1, Other: 0). The website is responsive and scales with the browser when smaller than the size in the PSD - own implementation, no design given (5).

The website has to meet professional expectations - visually and technically.
